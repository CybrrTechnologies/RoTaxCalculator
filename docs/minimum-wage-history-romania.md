## Key Observations for Tax Calculator

### Years with Mid-Year Changes
The following years had **mid-year minimum wage increases**, which is relevant for CASS calculations:

1. **2024**: 3,300 RON (Jan-Jun) → 3,700 RON (Jul-Dec)
   - Average for CASS: (3,300×6 + 3,700×6) / 12 = **3,500 RON/month**

2. **2023**: 3,000 RON (Jan-Sep) → 3,300 RON (Oct-Dec)
   - Average for CASS: (3,000×9 + 3,300×3) / 12 = **3,075 RON/month**

3. **2015**: 975 RON (Jan-Jun) → 1,050 RON (Jul-Dec)
   - Average for CASS: (975×6 + 1,050×6) / 12 = **1,012.50 RON/month**

4. **2014**: 850 RON (Jan-Jun) → 900 RON (Jul-Dec)
   - Average for CASS: (850×6 + 900×6) / 12 = **875 RON/month**

5. **2013**: 750 RON (Jan-Jun) → 800 RON (Jul-Dec)
   - Average for CASS: (750×6 + 800×6) / 12 = **775 RON/month**

6. **2008**: 500 RON (Jan-Jun) → 540 RON (Jul-Dec)
   - Average for CASS: (500×6 + 540×6) / 12 = **520 RON/month**

### Years with Uniform Minimum Wage
These years had a **single minimum wage** throughout the entire year:
- 2025, 2022, 2021, 2020, 2019, 2018, 2017, 2016, 2012, 2011, 2010, 2009, 2007

## Implications for CASS Calculation

When a year has mid-year minimum wage changes, the question arises:

**Does ANAF use:**
1. ❓ Weighted average of both values (e.g., 3,500 for 2024)?
2. ❓ The end-of-year value (e.g., 3,700 for 2024)?
3. ❓ The start-of-year value (e.g., 3,300 for 2024)?
